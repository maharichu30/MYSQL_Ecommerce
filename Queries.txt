Question1:Retrieve all customers who have placed an order in the last 30 days

Answer:
SELECT DISTINCT c.*
FROM customers c
JOIN orders o ON c.id = o.customer_id
WHERE o.order_date >= CURDATE() - INTERVAL 30 DAY;
output:
1	Alice Johnson	alice@example.com	123 Apple St
2	Bob Smith	    bob@example.com	    456 Banana Ave

Question2:Get the total amount of all orders placed by each customer.

Answer:
SELECT c.name, SUM(o.total_amount) AS total_spent
FROM customers c
JOIN orders o ON c.id = o.customer_id
GROUP BY c.id;
Output:
Alice Johnson	200.00
Bob Smith	    200.00

Question5:Retrieve the top 3 products with the highest price.

Answer:
SELECT id, name, price, discount, (price - discount) AS effective_price, description
FROM products
ORDER BY effective_price DESC
LIMIT 3;


output:
3	Product C	45.00	Description for Product C	0.00
2	Product B	35.00	Description for Product B	0.00
1	Product A	25.00	Description for Product A	0.00

Question6:Get the names of customers who have ordered Product A.

Answer:
SELECT c.name
FROM customers c
WHERE EXISTS (
    SELECT 1
    FROM orders o
    JOIN order_items oi ON o.id = oi.order_id
    JOIN products p ON oi.product_id = p.id
    WHERE o.customer_id = c.id
      AND p.name = 'Product A'
);


Output:
Alice Johnson

Question7:Join the orders and customers tables to retrieve the customer's name and order date for each order.

Answer:
SELECT c.name, o.order_date
FROM orders o
JOIN customers c ON o.customer_id = c.id;

Question9:Normalize the database by creating a separate table for order items and updating the orders table to reference the order_items table.

Answer:
CREATE TABLE order_items (
    id INT AUTO_INCREMENT PRIMARY KEY,
    order_id INT,
    product_id INT,
    quantity INT DEFAULT 1,
    FOREIGN KEY (order_id) REFERENCES orders(id),
    FOREIGN KEY (product_id) REFERENCES products(id)
);
